# Interactive Car List

Aplicação em JavaScript que percorre uma lista de carros, exibindo cada item de forma sequencial com interação do usuário para continuar ou interromper a execução.

## 🚀 Funcionalidades
- Exibição sequencial de itens com `alert`
- Controle de fluxo com `while`
- Interação do usuário com `confirm`
- Possibilidade de interromper a execução a qualquer momento

## 🛠 Tecnologias
- HTML
- JavaScript
